PumilioR

R package to query and get data out of a Pumilio system

Download source .tar.gz

View on GitHub

Functions to get data from a Pumilio system. Requires version 2.6.0, or newer, of Pumilio installed and XML turned on.

The package allows you to search a Pumilio archive and download specific sound files.

pumilioR version 1.3 is available on CRAN: http://cran.r-project.org/web/packages/pumilioR/index.html

Vignette with setup instructions and examples.
